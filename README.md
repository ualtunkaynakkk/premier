# PC Premier 3A2Ö — Penti Performans Paneli

Bölge müdürleri ve mağaza bazında HGO (Hedef Gerçekleşme Oranı) performansını görselleştiren interaktif dashboard.

## Özellikler
- Bölge müdürü bazında podyum sıralaması
- Mağaza detay tablosu (HGO bazlı sıralı)
- BM'ye göre filtreleme
- Mağaza adına göre arama
